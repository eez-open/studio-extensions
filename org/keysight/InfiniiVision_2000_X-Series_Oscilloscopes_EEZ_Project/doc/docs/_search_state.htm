General :SEARch Commands

Name

:SEARch:STATe —

Command Syntax

:SEARch:STATe <value>
<value> ::= {{0 | OFF} | {1 | ON}}

The :SEARch:STATe command enables or disables the search feature.

Query Syntax

:SEARch:STATe?

The :SEARch:STATe? query returns returns the current setting.

Return Format

<value><NL>
<value> ::= {0 | 1}
