:MEASure Commands

Name

:MEASure:SHOW —

Command Syntax

:MEASure:SHOW <show>
<show> ::= {1 | ON}

The :MEASure:SHOW command enables markers for tracking measurements on the display. This feature is always on.

Query Syntax

:MEASure:SHOW?

The :MEASure:SHOW? query returns the current state of the markers.

Return Format

<show><NL>
<show> ::= 1
