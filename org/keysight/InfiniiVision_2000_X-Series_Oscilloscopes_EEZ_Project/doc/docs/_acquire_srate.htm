:ACQuire Commands

Name

:ACQuire:SRATe —

Query Syntax

:ACQuire:SRATe?

The :ACQuire:SRATe? query returns the current oscilloscope acquisition sample rate. The sample rate is not directly controllable.

Return Format

<sample_rate><NL>
<sample_rate> ::= sample rate in NR3 format
