:CALibrate Commands

Name

:CALibrate:TEMPerature —

Query Syntax

:CALibrate:TEMPerature?

The :CALibrate:TEMPerature? query returns the change in temperature since the last user calibration procedure.

Return Format

<return value><NL>
<return value> ::= degrees C delta since last cal in NR3 format
