:SBUS<n>:CAN Commands

Name

:SBUS<n>:CAN:COUNt:TOTal —

Query Syntax

:SBUS<n>:CAN:COUNt:TOTal?

Returns the total frame count.

Return Format

<frame_count><NL>
<frame_count> ::= integer in NR1 format

Errors

  • -241, Hardware missing
