Overload Event Register (:OVLRegister)

The Overload Event Register register hosts these bits:

NameLocationDescription
Channel 1 Faultbit 6Fault has occurred on Channel 1 input.
Channel 2 Faultbit 7Fault has occurred on Channel 2 input.
Channel 3 Faultbit 8Fault has occurred on Channel 3 input.
Channel 4 Faultbit 9Fault has occurred on Channel 4 input.
External Trigger Faultbit 10Fault has occurred on External Trigger input.
