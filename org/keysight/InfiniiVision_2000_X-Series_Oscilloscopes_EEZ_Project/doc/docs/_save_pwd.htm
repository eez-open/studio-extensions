:SAVE Commands

Name

:SAVE:PWD —

Command Syntax

:SAVE:PWD <path_name>
<path_name> ::= quoted ASCII string

The :SAVE:PWD command sets the present working directory for save operations.

Query Syntax

:SAVE:PWD?

The :SAVE:PWD? query returns the currently set working directory for save operations.

Return Format

<path_name><NL>
<path_name> ::= quoted ASCII string
