:MTESt Commands

Name

:MTESt:COUNt:TIME —

Query Syntax

:MTESt:COUNt:TIME?

The :MTESt:COUNt:TIME? query returns the elapsed time in the current mask test run.

Return Format

<time><NL>
<time> ::= elapsed seconds in NR3 format.
