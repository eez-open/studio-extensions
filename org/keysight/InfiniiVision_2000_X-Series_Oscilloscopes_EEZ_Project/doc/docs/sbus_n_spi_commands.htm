:SBUS<n> Commands

:SBUS<n>:SPI Commands

These commands are only valid when the low-speed IIC and SPI serial decode option (Option LSS) has been licensed.

:SBUS<n>:SPI Commands Summary:

CommandQueryOptions and Query Returns
:SBUS<n>:SPI:BITorder <order>:SBUS<n>:SPI:BITorder?

<order> ::= {LSBFirst | MSBFirst}

:SBUS<n>:SPI:CLOCk:SLOPe <slope>:SBUS<n>:SPI:CLOCk:SLOPe?<slope> ::= {NEGative | POSitive}
:SBUS<n>:SPI:CLOCk:TIMeout <time_value>:SBUS<n>:SPI:CLOCk:TIMeout?<time_value> ::= time in seconds in NR3 format
:SBUS<n>:SPI:FRAMing <value>:SBUS<n>:SPI:FRAMing?<value> ::= {CHIPselect | {NCHipselect | NOTC} | TIMeout}
:SBUS<n>:SPI:SOURce:CLOCk <source>:SBUS<n>:SPI:SOURce:CLOCk?

<value> ::= {CHANnel<n> | EXTernal} for the DSO models

<value> ::= {CHANnel<n> | DIGital<d>} for the MSO models

<n> ::= 1 to (# analog channels) in NR1 format

<d> ::= 0 to (# digital channels - 1) in NR1 format
