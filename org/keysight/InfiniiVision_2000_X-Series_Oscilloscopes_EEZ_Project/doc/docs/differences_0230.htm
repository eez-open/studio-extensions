What's New

What's New in Version 2.30

New features in version 2.30 of the InfiniiVision 2000 X-Series oscilloscope software are:

  • Support for CAN/LIN, I2C/SPI, and UART/RS232 serial triggering and decode.

  • Saving Multi Channel Waveform data (*.h5) format files that can be opened by the N8900A InfiniiView oscilloscope analysis software.

More detailed descriptions of the new and changed commands appear below.

New Commands

CommandDifferences

:LISTer Commands

Commands for turning on/off the serial decode Lister display and returning data from the Lister display.

:SAVE:LISTer[:STARt]

Saves the Lister display data to a file.

:SAVE:MULTi[:STARt]

Lets you save Multi Channel Waveform data (*.h5) format files that can be opened by the N8900A InfiniiView oscilloscope analysis software.

:SBUS<n>:CAN Commands

Commands for CAN triggering and serial decode.

:SBUS<n>:DISPlay

Turns serial decode ON or OFF.

:SBUS<n>:IIC Commands

Commands for IIC triggering and serial decode.

:SBUS<n>:LIN Commands

Commands for LIN triggering and serial decode.

:SBUS<n>:MODE

Selects the type of serial decode.

:SBUS<n>:SPI Commands

Commands for SPI triggering and serial decode.

:SBUS<n>:UART Commands

Commands for UART/RS232 triggering and serial decode.

:SEARch:COUNt

Returns the number of search events found.

:SEARch:MODE

Can select SERial1.

:SEARch:SERial:CAN Commands

Commands for finding CAN events in the captured data.

:SEARch:SERial:IIC Commands

Commands for finding IIC events in the captured data.

:SEARch:SERial:LIN Commands

Commands for finding LIN events in the captured data.

:SEARch:SERial:SPI Commands

Commands for finding SPI events in the captured data.

:SEARch:SERial:UART Commands

Commands for finding UART/RS232 events in the captured data.

:SEARch:STATe

Enables or disables the search feature.

:TRIGger:LEVel:ASETup

Sets the trigger levels of all displayed analog channels to their waveforms' 50% values.

Changed Commands

CommandDifferences

:BLANk

You can now use the SBUS1 source parameter to turn off the serial decode display.

:DIGitize

You can now use the SBUS1 source parameter to acquire the serial waveform.

:SAVE:WAVeform:FORMat

The ALB format is no longer supported.

:STATus

You can now use the SBUS1 source parameter to view the serial decode display status.

:SYSTem:MENU

The LISTer parameter is now available.

:VIEW

You can now use the SBUS1 source parameter to turn on the serial decode display.

:WAVeform:SOURce

Can select SBUS1 as the waveform source.

:WAVeform:SOURce:SUBSource

With the SPI and UART/RS232 serial decode options, subsources are now valid in the 2000 X-Series oscilloscopes.
