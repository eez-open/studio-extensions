:TRIGger:TV Commands

Name

:TRIGger:TV:POLarity —

Command Syntax

:TRIGger:TV:POLarity <polarity>
<polarity> ::= {POSitive | NEGative}

The :TRIGger:TV:POLarity command sets the polarity for the TV trigger.

Query Syntax

:TRIGger:TV:POLarity?

The :TRIGger:TV:POLarity? query returns the TV trigger polarity.

Return Format

<polarity><NL>
<polarity> ::= {POS | NEG}
