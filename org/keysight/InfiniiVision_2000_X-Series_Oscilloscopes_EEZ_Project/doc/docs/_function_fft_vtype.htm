:FUNCtion Commands

Name

:FUNCtion[:FFT]:VTYPe —

Command Syntax

:FUNCtion[:FFT]:VTYPe <units>
<units> ::= {DECibel | VRMS}

The :FUNCtion[:FFT]:VTYPe command specifies FFT vertical units as DECibel or VRMS.

Query Syntax

:FUNCtion[:FFT]:VTYPe?

The :FUNCtion[:FFT]:VTYPe? query returns the current FFT vertical units.

Return Format

<units><NL>
<units> ::= {DEC | VRMS}
