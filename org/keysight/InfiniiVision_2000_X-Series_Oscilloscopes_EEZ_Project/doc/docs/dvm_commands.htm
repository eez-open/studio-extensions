Agilent InfiniiVision 2000 X-Series Oscilloscopes Programmer's Guide

:DVM Commands

When the optional DSOXDVM digital voltmeter analysis feature is licensed, these commands control the digital voltmeter (DVM) feature.

:DVM Commands Summary:

CommandQueryOptions and Query Returns
:DVM:ARANge {{0 | OFF} | {1 | ON}}:DVM:ARANge?{0 | 1}
n/a:DVM:CURRent?

<dvm_value> ::= floating-point number in NR3 format

:DVM:ENABle {{0 | OFF} | {1 | ON}}:DVM:ENABle?{0 | 1}
n/a:DVM:FREQuency?

<freq_value> ::= floating-point number in NR3 format

:DVM:MODE <mode>:DVM:MODE?

<dvm_mode> ::= {ACRMs | DC | DCRMs | FREQuency}

:DVM:SOURce <source>:DVM:SOURce?

<source> ::= {CHANnel<n>}

<n> ::= 1-2 or 1-4 in NR1 format
