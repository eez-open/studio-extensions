:SAVE Commands

Name

:SAVE:IMAGe:PALette —

Command Syntax

:SAVE:IMAGe:PALette <palette>
<palette> ::= {COLor | GRAYscale}

The :SAVE:IMAGe:PALette command sets the image palette color.

Query Syntax

:SAVE:IMAGe:PALette?

The :SAVE:IMAGe:PALette? query returns the selected image palette color.

Return Format

<palette><NL>
<palette> ::= {COL | GRAY}
