:DVM Commands

Name

:DVM:ENABle —

Command Syntax

:DVM:ENABle <setting>
<setting> ::= {{OFF | 0} | {ON | 1}}

The :DVM:ENABle command turns the digital voltmeter (DVM) analysis feature on or off.

Query Syntax

:DVM:ENABle?

The :DVM:ENABle? query returns a flag indicating whether the digital voltmeter (DVM) analysis feature is on or off.

Return Format

<setting><NL>
<setting> ::= {0 | 1}
