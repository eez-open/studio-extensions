:SBUS<n>:UART Commands

Name

:SBUS<n>:UART:COUNt:TXFRames —

Query Syntax

:SBUS<n>:UART:COUNt:TXFRames?

Returns the UART Tx frame count.

Return Format

<frame_count><NL>
<frame_count> ::= integer in NR1 format

Errors

  • -241, Hardware missing
