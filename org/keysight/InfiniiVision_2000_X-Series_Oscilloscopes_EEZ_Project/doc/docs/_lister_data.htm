:LISTer Commands

Name

:LISTer:DATA —

Query Syntax

:LISTer:DATA?

The :LISTer:DATA? query returns the lister data.

Return Format

<binary block><NL>
<binary_block> ::= comma-separated data with newlines at the
                   end of each row
