Root (:) Commands

Name

:MTERegister[:EVENt] (Mask Test Event Event Register) —

Query Syntax

:MTERegister[:EVENt]?

The :MTERegister[:EVENt]? query returns the integer value contained in the Mask Test Event Event Register and clears the register.

Mask Test Event Event Register:

BitNameDescriptionWhen Set (1 = High = True), Indicates:
15-11------(Not used.)
10Auto MaskAuto Mask CreatedAuto mask creation completed.
9------(Not used.)
8StartedMask Testing StartedMask testing started.
7-2------(Not used.)
1FailMask Test FailThe mask test failed.
0CompleteMask Test CompleteThe mask test is complete.

Return Format

<value><NL>
<value> ::= integer in NR1 format.
