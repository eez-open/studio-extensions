:DVM Commands

Name

:DVM:FREQuency —

Query Syntax

:DVM:FREQuency?

The :DVM:FREQuency? query returns the displayed 5-digit frequency value that is displayed below the main DVM value.

Return Format

<freq_value><NL>
<freq_value> ::= floating-point number in NR3 format
