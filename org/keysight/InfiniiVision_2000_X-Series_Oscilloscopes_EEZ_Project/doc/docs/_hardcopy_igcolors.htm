Obsolete and Discontinued Commands

Name

:HARDcopy:IGColors —

Command Syntax

:HARDcopy:IGColors <value>
<value> ::= {{OFF | 0} | {ON | 1}}

The HARDcopy:IGColors command controls whether the graticule colors are inverted or not.

The :HARDcopy:IGColors command is an obsolete command provided for compatibility to previous oscilloscopes. Use the :HARDcopy:INKSaver command instead.

Query Syntax

:HARDcopy:IGColors?

The :HARDcopy:IGColors? query returns a flag indicating whether graticule colors are inverted or not.

Return Format

<value><NL>
<value> ::= {0 | 1}
