Agilent InfiniiVision 2000 X-Series Oscilloscopes Programmer's Guide

:WMEMory<r> Commands

Control reference waveforms.

:WMEMory<r> Commands Summary:

CommandQueryOptions and Query Returns
:WMEMory<r>:CLEarn/a

<r> ::= 1-2 in NR1 format

:WMEMory<r>:DISPlay {{0 | OFF} | {1 | ON}}:WMEMory<r>:DISPlay?

<r> ::= 1-2 in NR1 format

{0 | 1}

:WMEMory<r>:LABel <string>:WMEMory<r>:LABel?

<r> ::= 1-2 in NR1 format

<string> ::= any series of 10 or less ASCII characters enclosed in quotation marks

:WMEMory<r>:SAVE <source>n/a

<r> ::= 1-2 in NR1 format

<source> ::= {CHANnel<n> | FUNCtion | MATH}

<n> ::= 1 to (# analog channels) in NR1 format

NOTE: Only ADD or SUBtract math operations can be saved as reference waveforms.

:WMEMory<r>:SKEW <skew>:WMEMory<r>:SKEW?

<r> ::= 1-2 in NR1 format

<skew> ::= time in seconds in NR3 format

:WMEMory<r>:YOFFset <offset>[suffix]:WMEMory<r>:YOFFset?

<r> ::= 1-2 in NR1 format

<offset> ::= vertical offset value in NR3 format

[suffix] ::= {V | mV}

:WMEMory<r>:YRANge <range>[suffix]:WMEMory<r>:YRANge?

<r> ::= 1-2 in NR1 format

<range> ::= vertical full-scale range value in NR3 format

[suffix] ::= {V | mV}

:WMEMory<r>:YSCale <scale>[suffix]:WMEMory<r>:YSCale?

<r> ::= 1-2 in NR1 format

<scale> ::= vertical units per division value in NR3 format

[suffix] ::= {V | mV}
