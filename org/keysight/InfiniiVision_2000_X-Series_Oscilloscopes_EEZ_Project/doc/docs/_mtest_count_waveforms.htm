:MTESt Commands

Name

:MTESt:COUNt:WAVeforms —

Query Syntax

:MTESt:COUNt:WAVeforms?

The :MTESt:COUNt:WAVeforms? query returns the total number of waveforms acquired in the current mask test run.

Return Format

<count><NL>
<count> ::= number of waveforms in NR1 format.
