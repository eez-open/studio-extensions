:HARDcopy Commands

Name

:HARDcopy:AREA —

Command Syntax

:HARDcopy:AREA <area>
<area> ::= SCReen

The :HARDcopy:AREA command controls what part of the display area is printed. Currently, the only legal choice is SCReen.

Query Syntax

:HARDcopy:AREA?

The :HARDcopy:AREA? query returns the selected display area.

Return Format

<area><NL>
<area> ::= SCR
