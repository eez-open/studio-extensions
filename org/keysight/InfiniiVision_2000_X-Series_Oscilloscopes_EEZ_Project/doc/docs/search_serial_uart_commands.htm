:SEARch:SERial:UART Commands Summary:

CommandQueryOptions and Query Returns
:SEARch:SERial:UART:DATA <value>:SEARch:SERial:UART:DATA?

<value> ::= 8-bit integer from 0-255 (0x00-0xff) in decimal, <hexadecimal>, <binary>, or <quoted_string> format

<hexadecimal> ::= #Hnn where n ::= {0,..,9| A,..,F} for hexadecimal

<binary> ::= #Bnn...n where n ::= {0 | 1} for binary

<quoted_string> ::= any of the 128 valid 7-bit ASCII characters (or standard abbreviations)

:SEARch:SERial:UART:MODE <value>:SEARch:SERial:UART:MODE?<value> ::= {RDATa | RD1 | RD0 | RDX | TDATa | TD1 | TD0 | TDX | PARityerror | AERRor}
:SEARch:SERial:UART:QUALifier <value>:SEARch:SERial:UART:QUALifier?<value> ::= {EQUal | NOTequal | GREaterthan | LESSthan}
