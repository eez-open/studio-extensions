:WGEN Commands

Name

:WGEN:OUTPut —

Command Syntax

:WGEN:OUTPut <on_off>
<on_off> ::= {{1 | ON} | {0 | OFF}

The :WGEN:OUTPut command specifies whether the waveform generator signal output is ON (1) or OFF (0).

Query Syntax

:WGEN:OUTPut?

The :WGEN:OUTPut? query returns the current state of the waveform generator output setting.

Return Format

<on_off><NL>
<on_off> ::= {1 | 0}
