:DVM Commands

Name

:DVM:CURRent —

Query Syntax

:DVM:CURRent?

The :DVM:CURRent? query returns the displayed 3-digit DVM value based on the current mode.

Return Format

<dvm_value><NL>
<dvm_value> ::= floating-point number in NR3 format
