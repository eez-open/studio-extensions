:HARDcopy Commands

Name

:HARDcopy:LAYout —

Command Syntax

:HARDcopy:LAYout <layout>
<layout> ::= {LANDscape | PORTrait}

The :HARDcopy:LAYout command sets the hardcopy layout mode.

Query Syntax

:HARDcopy:LAYout?

The :HARDcopy:LAYout? query returns the selected hardcopy layout mode.

Return Format

<layout><NL>
<layout> ::= {LAND | PORT}
